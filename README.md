
# Библиотека для работы с API Московской Электронной Школы (МЭШ)

**Важно**: Оригинальный проект более не поддерживается. Данный форк поддерживается сообществом.

## О проекте

Официальное API Московской Электронной Школы (МЭШ) было закрыто в 2017 году. Однако мобильные и веб-приложения продолжают функционировать, взаимодействуя с сервером через внутренние, недокументированные эндпоинты.

Цель данного проекта — обратная разработка (reverse engineering) этих эндпоинтов для создания неофициальной Python библиотеки, упрощающей доступ к данным МЭШ.

## Методология

Для перехвата и анализа трафика между клиентским приложением и серверами МЭШ используется инструмент Burp Suite, функционирующий как прокси-сервер. Это позволяет фиксировать все запросы и ответы, идентифицируя используемые эндпоинты.

## Технические детали

- Бэкенд МЭШ реализован на Java с использованием Spring Framework (определено по структуре ошибок).
- Фронтенд написан на React.
