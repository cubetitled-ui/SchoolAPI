
# Библиотека для работы с API Московской Электронной Школы (МЭШ)

> ⚠️ **Важно**: Оригинальный проект больше не поддерживается. Этот форк активно развивается сообществом.

## Что такое МЭШ и зачем нужна эта библиотека?

[Московская Электронная Школа (МЭШ)](https://school.mos.ru/) — это цифровая экосистема для образования, включающая электронный дневник, библиотеку материалов и другие сервисы. Однако у МЭШ нет публичного API — его закрыли в 2017 году из-за большого количества сторонних приложений.

**Но есть нюанс:** мобильное приложение и веб-версия дневника продолжают работать. А это значит, что они общаются с сервером через закрытые API-эндпоинты. Наша задача — найти эти эндпоинты и сделать их доступными для разработчиков.

## Цель проекта

Мы исследуем трафик мобильного приложения МЭШ, чтобы:
1. Выявить все используемые API-эндпоинты
2. Составить подробную документацию по работе с ними
3. Создать удобную Python-библиотеку для взаимодействия с API

## Инструменты

Для анализа трафика мы используем **Burp Suite** — профессиональный инструмент для тестирования безопасности веб-приложений. Он работает как прокси-сервер, перехватывая все запросы между приложением и сервером.

## Технический стек МЭШ

- **Бэкенд**: Java (Spring Framework) — определено по характерным ошибкам
- **Фронтенд**: React — видно в инструментах разработчика

```bash

```
